# Meal Planner

A mobile-first meal planning app for families.

## Features (Planned)

- Weekly meal suggestions
- Shopping list generation
- Recipe favorites
- ShopRite cart integration (future)

## Tech Stack

- SvelteKit
- TypeScript
- Supabase
- Spoonacular API

## Development

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```
